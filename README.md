# oiline!!!

## サイト概要

料理の用途に合わせて様々な油を紹介し、ブログ形式で共有できるサイトです。

### サイトテーマ

食材や料理によって油を使い分ける楽しさ

### テーマを選んだ理由

クックパッドはレシピですが、その手前に「油」から知る事で、更に料理が楽しくなるアプリを作りたかったからです
